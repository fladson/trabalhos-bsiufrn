Como o código é na linguagem Ruby, é preciso tê-la devidamente instalada.

Antes de rodar o código é preciso instalar uma biblioteca que usei para auxiliar nas requisições das entradas do terminal.

Para isso apenas rode os seguinte comando.

cd ftp_fladson_kelvin/

cd highline/

sudo ruby setup.rb 
ou 
ruby setup.rb


depois é só voltar para o diretório anterior 

cd ..

e chamar o código Ruby da seguinte forma: 
 
ruby ftp.rb host.name.com 21 
OU  
ruby ftp.rb host.name.com
